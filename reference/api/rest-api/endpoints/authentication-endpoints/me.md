# Me

Quick information about the authenticated user.

| URL          | Requires Auth | HTTP Method |
| ------------ | ------------- | ----------- |
| `/api/v1/me` | `yes`         | `GET`       |

## Example Call

```bash
curl -H "X-Auth-Token: 9HqLlyZOugoStsXCUfD_0YdwnNnunAJF8V47U3QHXSq" \
     -H "X-User-Id: aobEdbYhXfu5hkeqG" \
     http://localhost:3000/api/v1/me
```
